# Hackathon_Hairloss-project

#### 2024 빅데이터/AI 해커톤 탈모 셀프 점검 프로젝트 - 장려상 :trophy:

## 문제 정의
건강보험관리공단에 따르면 국내 탈모 인구가 10000만명으로 추산되고 10~30대 젊은 탈모 환자가 전체의 50%를 차지하고 있을 정도로 심각한 탈모화 시대
그에 따라 유튜브, SNS 등에서 자가 탈모 진단 방법 등이 큰 인기를 끌고 있다.
